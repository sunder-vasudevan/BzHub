# BzHub Help

Welcome to BzHub. This help file is context‑sensitive in the app. Use the Help button (top bar) or press F1.

---

## Dashboard
- Review sales and inventory KPIs.
- Use the projection window to switch between 7/30/quarter/year.
- Double‑click charts to zoom; single‑click to close.

---

## CRM
### Inventory (CRM)
- Add, update, or remove inventory items.
- Use the image field to attach product visuals.
- Keep quantities above thresholds to avoid low‑stock alerts.

### POS (CRM)
- Search for items and add to cart.
- Update quantities before checkout.
- Print the latest receipt from the POS tools.

### Reports (CRM)
- View sales summaries and trends.
- Use the date range selector to adjust reporting windows.

### Bills (CRM)
- Review bills timeline entries.
- Print any selected sale or bill from the list.

### Visitors (CRM)
- Manage visitor records.
- Use cards to quickly review contact details.

---

## HR
### Employees (HR)
- Create employee profiles and manage status.
- Print employee ID cards from the action buttons.

### Payroll (HR)
- Add payroll records and update payment status.
- Use preview/print options where available.

### Appraisals (HR)
- Run the appraisal workflow: Self → 360 → Manager → Final.
- Use the list to open and update existing cycles.

### Feedback (HR)
- Create 360 feedback requests.
- Submit feedback entries at any time.

---

## Settings
- Company information is locked by default.
- Use the admin unlock action to edit sensitive fields.

---

## Safety & Confirmations
- Deletions and deactivations require a caution window and a confirmation.

---

## Shortcuts
- F1: Open context help
